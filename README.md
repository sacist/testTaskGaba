# Запуск проекта

## 1. Клонировать репозиторий
```
git clone <repo-url>
cd <project-folder>
```

## 2. Установить зависимости
```
npm install
```

## 3. Создать базу данных PostgreSQL
Создайте пустую базу данных в PostgreSQL.

## 4. Создать файл `.env`
В корне проекта создайте файл `.env` и укажите переменные окружения:

```
PG_CONNECTION=postgres://user:password@localhost:5432/db_name
PORT=3000
```

## 5. Создать таблицы
```
npm run create-tables
```

## 6. Запустить сервер
```
npm run start
```

# Disclaimer

В рамках тестового задания намеренно не использовались:

- ORM  
- Redis  
- усложнённая архитектура  
- базовые классы  
- кастомные error handlers  

Это сделано осознанно — я считаю, для подобного задания приоритетом является простота, читаемость и скорость понимания кода.

При необходимости проект легко масштабируется и может быть расширен с использованием:

- ORM (например Prisma / TypeORM)  
- кеширования (Redis)  
- слоистой архитектуры  
- централизованной обработки ошибок
